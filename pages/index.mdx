## Enough About Me

I love teaching programming and am always curious how to do it better. I'm especially interested in how we can better translate evidence-based teaching/learning strategies into tools, materials and guides that are actually useful for teachers and learners. Over the last 7 years I've been doing just that; running and designing programming courses from the ground up inspired by my students & fellow teachers, computing education research, and by the tight constraints placed on small schools and non-profits.

I am also curious how we can develop content authoring/generation tools, class management tools, and study tools that make it realistic for learners & small organizations to manage their own curriculum without sacrificing pedagogical value. I call this an _Ownable Code School_ when everything necessary to run a programming course (with or without teachers) can be set up by an alumni with only the skills learned in the course (more about this in my notes). So far we have a couple functioning prototypes but could use some help making them more reliable.

If any of this sounds fun to you, [send me an email](#send-evan-an-email). We can probably find something to work on together.
